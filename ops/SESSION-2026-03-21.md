## Changes Implemented

### Registry trust hardening

- Updated [`automation/memory-api/app/registry_service.py`](/Users/sgkmeyer/ai-automation-stack/automation/memory-api/app/registry_service.py):
  - scheme-equivalent web URLs are now treated as duplicate candidates
  - capture now actively converges existing `http`/`https` duplicates instead of reinforcing them
  - merge behavior preserves capture history and merged metadata more carefully
  - blocked pages now degrade to fallback registry items with:
    - `processing_status=ready`
    - `metadata.extraction_mode=blocked_metadata_only`
    - URL-derived title
    - note-aware `why_it_matters`
  - true hard failure is now reserved for cases where no usable record can be formed
